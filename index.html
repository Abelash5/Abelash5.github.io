Java Game: Fruit Ninja

My first programming project is a java 2d game. It is a simple game of a ninja moving through the levels, avoiding the interference of the enemies to complete the game. There are certain parameters to move to the next level such as the ninja has to collect all the bananas in the current level to advance to the next level. Each level get harder where the amount of enemies/obstructions increases as you progress. Also there are three different weapons the ninja can use which can be functioned by the key pressed type. The ninja needs to avoid being killed by the enemies and the obstructions such as lava. The major challenges I faced was I found it difficult to recognize how my errors was getting caused and took a significant amount of time to rectify it. Lessons learnt was I'm able to make a clean efficient code, by using encapsulation or general java comments
